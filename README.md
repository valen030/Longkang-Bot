
# LKG Discord Music Bot

A simple discord bot with the sole purpose of playing music by using Yucked/Victoria library and Lavalink.


## Quick Setup

### Currently for developers 
    1) Clone the repositories
    2) Add discord bot token in appsettings.json
    3) Rebuild project in release mode and run it
    4) A log file is produced in Server folder in release mode
    5) Look for a link which need you to verifed device (youtube required), verify with your non-main account
    6) Back to the log file there will be a refreshToken
    7) Update refreshToken in application.yml
